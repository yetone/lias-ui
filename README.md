# lias-ui

WARNING: This project is still in the very early stages of development.

## Install

```bash
yarn add lias-ui
```

## Usage

```tsx
import React from 'react'
import { Button } from 'lias-ui'

export default function App() {
  return <Button type='primary'>Hello World!</Button>
}
```
